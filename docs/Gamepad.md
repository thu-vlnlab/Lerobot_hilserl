  Gamepad 控制方式
  | 按钮/摇杆         | 功能         |
  |-------------------|--------------|
  | 左摇杆            | X-Y 平面移动 |
  | 右摇杆（上下）    | Z 方向移动   |
  | Back（左小按钮）  | 闭合夹爪     |
  | Start（右小按钮） | 打开夹爪     |
  | RB                | 干预模式开关 |
  | A                 | 标记失败     |
  | B                 | 退出         |
  | X                 | 重录 episode |
  | Y                 | 标记成功     |
